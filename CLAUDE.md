# Homepage Veniaminch

Личный сайт + оргструктура ресторана «Высоко» + дашборд экосистемы. URL: veniaminch.ru

## Стек
- **Frontend:** Чистый HTML5 + CSS3 + минимальный JS (без сборщиков)
- **Сервер:** nginx Alpine (Docker), шрифт DM Sans

## Страницы
- `index.html` — дашборд экосистемы: карточки проектов (LOC, архитектура, стоимость), статистика токенов
- `org.html` (~970 строк) — оргструктура: иерархия, зал/кухня, 18 функциональных зон, матрица взаимодействий 9x9
- `infra.html` — инфраструктура: серверы, сервисы, схема деплоя

## Дизайн
- index.html: современный дашборд с карточками проектов и метриками
- org.html: тёплая светлая тема (#f0ebe4 фон, #b07d4f акцент), карточки с тенями
- Sidebar навигация (сбоку на десктопе, снизу на мобильных)
- Раскрывающиеся описания должностей, адаптив (900px breakpoint)

## Деплой
Docker (nginx:alpine) + GitHub (veniaminch-sudo/homepage-veniaminch) → Coolify → veniaminch.ru
